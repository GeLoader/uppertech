FAQs

What are data analytics?

Data analytics is the process of collecting, cleaning, and analyzing data to extract meaningful insights. It is a multidisciplinary field that uses a wide range of techniques, including statistics, computer science, and mathematics.

What is data mining?

Data mining is the process of extracting knowledge and insights from large datasets. It is a subfield of computer science and statistics that uses a variety of techniques to identify patterns, trends, and relationships in data. Data mining can be used to solve a wide range of problems

What is exploratory data analysis (EDA)?

Exploratory data analysis (EDA) is a process of investigating data to discover patterns, identify outliers, and test hypotheses. It is an important step in any data science project, as it helps to ensure that the data is well-understood and that the right tools and techniques are used for subsequent analysis.

What is confirmatory data analysis (CDA)?

Confirmatory data analysis (CDA) is a statistical process that tests specific hypotheses about a dataset. It is typically used to confirm or reject findings from exploratory data analysis (EDA).

What are predictive analytics?

Predictive analytics is a type of data analytics that uses historical data to predict future outcomes. It uses statistical models and machine learning algorithms to identify patterns in data and to make predictions about the future. Predictive analytics can be used in a wide range of industries, including business, finance, healthcare, and government.

What is data visualisation

Data visualization is the process of communicating data through visual elements, such as charts, graphs, and maps. It is a powerful tool for communicating complex information in a way that is easy to understand and interpret.
